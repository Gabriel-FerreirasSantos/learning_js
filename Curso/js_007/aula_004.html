Operadores Aritméticos

Adição = +

Subtração = -

Multiplicação = *

Divisão = /

Resto da divisão inteira = %

Potenciação = **

Precedência de operadores

1° ()

2° **

3° *, /, %

4° + e -

Operadore de atribuição simples

var a = 5 + 3;

var b = a % 5;

var c = 5 * b ** 2;

var d = 10 - a / 2;

var e = 6 * 2 / d;

var f = b % e + 4 / e;

Observação: O = tem um significado de recebe e não que é igual. Exemplo:

var a recebe 5 + 3;

Operadores de auto-atribuição

var n = 3;

n = n + 4;

Observação: Aqui a variável n recebe 3, logo em seguida a variável n recebe ela mesma mais 4. Sendo assim ela deixa de ser 3 e agora vale 7


Observação2: Também pode ser usada da seguinte forma:

var n = 3;

n += 4;
