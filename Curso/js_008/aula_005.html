Relacionais

Maior >

Menor <

Maior ou igual >=

Menor ou igual <=

Igual ==

Diferente !=

Exemplo

5 > 2 = True

5 < 2 = False

5 >= 2 = True

5 <= 2 = False

5 == 2 = False

5 != 2 = True

Observação: Temos que ter cuidado com a Identidade de tal coisa. Exemplo:

5 == 5 = True;

5 == "5" = True;

É confuso ja que são de tipos diferentes, porém, o javascript olha a grandeza e não type

Operador de identidade restrita:

Porém tem um operador de identidade que é restrito a isso.

Usamos o ===.Exemplo:

Em vez de usar assim:

5 == "5" = True;

Usamos assim:

5 === "5" = False;

Agora o javascript irá ver se eles são identico tanto em tipo quanto a grandeza. Sendo assim

5 === 5 = True;

pois são do mesmo tipo e mesma grandeza.

estado == "RJ" || estado == "SP"; //o estado é RJ ou SP?

Operadores Ternário

? True

: False

teste lógico ? True : False

Exemplo:

média = 5.5;
média >= 7.0 ? "Aprovado" : "Reprovado";
